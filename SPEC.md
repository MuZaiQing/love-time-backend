# 情侣相恋时长 & 纪念日记录网页 - 规格文档

## 1. 概念与愿景

一款专为情侣设计的时间记录网页，记录相恋天数与重要纪念日。视觉上追求「静谧流光」美学——如同夜空中缓慢流淌的星光轨迹，营造温柔而不喧闹的氛围。整体风格克制优雅，以核心计时内容为视觉焦点，背景流光作为细腻装饰而非干扰。

## 2. 设计语言

### 美学方向
- **主题**: 静谧流光 (Serene Luminescence)
- **意象**: 夜空中缓慢流动的星光、黎明前最轻柔的天光
- **关键词**: 克制、优雅、温柔、专注

### 色彩系统

**浅色模式**:
- 主色: `#6366F1` (靛蓝紫)
- 次色: `#8B5CF6` (柔紫)
- 强调色: `#EC4899` (玫瑰粉)
- 背景: `#FAFAFE` (近白微紫)
- 卡片背景: `rgba(255, 255, 255, 0.85)`
- 文字主色: `#1E1B4B` (深靛)
- 文字次色: `#6B7280`
- 流光主色: `rgba(236, 72, 153, 0.15)` (淡粉)
- 流光次色: `rgba(139, 92, 246, 0.12)` (浅紫)

**深色模式**:
- 主色: `#818CF8` (亮靛蓝)
- 次色: `#A78BFA` (亮柔紫)
- 强调色: `#F472B6` (亮玫瑰粉)
- 背景: `#0F0E1A` (深夜蓝黑)
- 卡片背景: `rgba(30, 27, 75, 0.85)`
- 文字主色: `#F1F5F9` (近白)
- 文字次色: `#94A3B8`
- 流光主色: `rgba(165, 180, 252, 0.2)` (浅白淡青)
- 流光次色: `rgba(103, 232, 249, 0.15)` (淡青)

### 字体
- 主字体: `"Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif`
- 数字字体: `"DIN Alternate", "Roboto Mono", monospace`
- 字重: 300 (轻盈), 400 (正文), 500 (中等), 600 (标题)

### 空间系统
- 基础单位: 4px
- 间距: 8px, 12px, 16px, 24px, 32px, 48px, 64px
- 圆角: 8px (小), 16px (中), 24px (大), 32px (卡片)
- 最大内容宽度: 480px (移动端优先)

### 动效哲学
- **克制**: 动效服务于体验，不制造噪音
- **自然**: 缓动曲线模拟自然物理 (`cubic-bezier(0.4, 0, 0.2, 1)`)
- **缓慢**: 呼吸光晕周期 3-4 秒，流光移动平缓
- **细腻**: 微小缩放 + 阴影过渡，反馈克制

### 视觉资产
- 图标: Lucide Icons (线性风格，stroke-width: 1.5)
- 无外部图片依赖
- 装饰: CSS 渐变光晕、p5.js 流光轨迹

## 4. 功能与交互

### 核心功能

#### 4.1 用户认证 (Supabase)
- 匿名登录 (首次访问自动创建)
- 邮箱/手机号登录 (可选)
- 登录状态持久化 (localStorage)
- 退出登录

#### 4.2 情侣绑定
- 创建绑定: 生成 6 位邀请码
- 加入绑定: 输入邀请码加入
- 显示双方信息 (昵称、头像占位)
- 绑定关系云端同步

#### 4.3 相恋时长计时
- 显示相恋总天数 (整数)
- 显示相识天数
- 显示在一起的小时数、分钟数 (实时更新)
- 开始日期可编辑

#### 4.4 纪念日管理
- CRUD 操作: 创建、读取、更新、删除
- 字段: 名称、日期、类型 (纪念日/生日/其他)
- 显示距离下一次的天数倒计时
- 列表按日期排序 (即将到来优先)
- 淡入加载动画

#### 4.5 深色/浅色模式
- 自动跟随系统偏好
- 支持手动切换
- 主题偏好持久化

#### 4.6 离线缓存
- Service Worker 缓存静态资源
- IndexedDB 缓存用户数据
- 网络恢复后自动同步

#### 4.7 数据同步
- 实时同步至 Supabase
- 冲突处理: 以最新时间戳为准
- 离线队列: 离线操作联网后同步

### 交互细节

#### 核心计时卡片
- 呼吸光晕: 3-4 秒完成明暗循环
- 数字变化: 平滑滚动动画 (类似里程表)
- 点击日期: 弹出编辑开始日期面板

#### 纪念日条目
- hover: `transform: scale(1.02)`, 阴影加深
- 点击: 展开详情或编辑面板
- 左滑/长按: 显示删除选项 (移动端)
- 删除确认: 二次确认弹窗

#### 新增/编辑面板
- 淡入淡出动画 (opacity 0→1, 200ms)
- 背景遮罩渐显
- 表单验证即时反馈

#### 空状态
- 纪念日列表为空: 优雅的空状态插画 (CSS 绘制)
- 引导文字: "添加你们的第一个纪念日"

## 6. 技术方案

### 前端架构
- **框架**: 纯 HTML + CSS + JavaScript (无框架依赖)
- **构建**: 单 HTML 文件，自包含
- **CDN**:
  - p5.js (背景动效)
  - Lucide Icons (图标)
  - Supabase JS SDK (后端)

### 数据模型

#### User
```javascript
{
  id: string,
  nickname: string,
  avatar_url: string | null,
  couple_id: string | null,
  created_at: timestamp
}
```

#### Couple
```javascript
{
  id: string,
  invite_code: string,
  user1_id: string,
  user2_id: string | null,
  start_date: date,
  created_at: timestamp
}
```

#### Anniversary
```javascript
{
  id: string,
  couple_id: string,
  name: string,
  date: date,
  type: 'anniversary' | 'birthday' | 'other',
  created_at: timestamp,
  updated_at: timestamp
}
```

### API 设计 (Supabase)

#### 认证
- `supabase.auth.signInAnonymously()` - 匿名登录
- `supabase.auth.signOut()` - 退出登录
- `supabase.auth.onAuthStateChange()` - 监听状态

#### 情侣
- `POST /couples` - 创建情侣关系
- `GET /couples/:id` - 获取情侣信息
- `PATCH /couples/:id` - 更新开始日期
- `POST /couples/join` - 通过邀请码加入

#### 纪念日
- `GET /anniversaries?couple_id=eq.x` - 获取列表
- `POST /anniversaries` - 创建
- `PATCH /anniversaries/:id` - 更新
- `DELETE /anniversaries/:id` - 删除

### 离线策略
- Service Worker: 缓存 HTML/CSS/JS
- IndexedDB: 存储用户数据、纪念日数据
- 同步队列: 离线操作记录，联网后执行

### 背景动效技术
- **库**: p5.js
- **算法**: 流场 (Flow Field) - Perlin 噪声驱动
- **渲染**: 超细柔光线条，低透明度，缓慢移动
- **性能**:
  - 控制粒子数量 (50-80)
  - 使用 `noStroke()` 优化
  - 低帧率运行 (15-20 FPS)
  - 移动端降级 (减少粒子数)

### 动效实现
- CSS `animation` + `@keyframes`: 呼吸光晕、淡入淡出
- CSS `transition`: hover 效果、状态切换
- JavaScript: 数字滚动动画 (requestAnimationFrame)
